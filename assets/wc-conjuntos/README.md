# Imagenes de conjuntos WC

En esta carpeta se suben las imagenes de cada conjunto (modelo de WC Completo) para mostrarlas en Herramientas > WC Completo.

- **Nombre del archivo**: usar el **codigo** del conjunto (ej. `serie-comfort.jpg`). Si el conjunto no tiene codigo, se puede usar el `id` (UUID) o configurar la URL en el Panel de Control.
- **Formatos**: JPG o PNG recomendado.
- La app usa por defecto la convencion: `assets/wc-conjuntos/{codigo}.jpg` (ruta relativa o URL segun despliegue).
